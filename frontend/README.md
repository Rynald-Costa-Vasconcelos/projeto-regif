# Frontend REGIF

Aplicacao React + Vite do projeto REGIF.

## Rodar em desenvolvimento local

```powershell
npm install
copy .env.example .env
npm run dev
```

Endereco padrao: `http://127.0.0.1:5173`

## Variaveis de ambiente

- `VITE_API_URL`: host da API (sem `/api` tambem funciona).

Exemplo:

```env
VITE_API_URL=http://127.0.0.1:3000
```
